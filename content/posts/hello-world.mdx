## 블로그를 시작하며

드디어 개인 블로그를 만들었습니다! **Next.js**와 **MDX**를 활용해서 만든 이 블로그에 앞으로 개발 경험과 일상의 생각들을 기록해 나갈 예정입니다.

## 기술 스택

이 블로그는 다음 기술들로 만들어졌습니다:

- **Next.js 15** — App Router 기반
- **Tailwind CSS** — 스타일링
- **MDX** — 마크다운 + JSX 콘텐츠 작성
- **gray-matter** — 프론트매터 파싱
- **Giscus** — GitHub Discussions 기반 댓글

## 코드 예시

```typescript
function greet(name: string) {
  return `안녕하세요, ${name}!`;
}

console.log(greet("세상"));
```
